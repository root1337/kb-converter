Step 1: Prerequisites

>  Install latest version of Python 3
    >  Install beautifulsoup4 by running command below as hdadmin

pip install bs4

>  Import the batch converter action for Adobe Acrobat

>  Open "KB conversion log.xlsx"
>  Copy list of KB IDs of KBs to be downloaded and converted
    >   20-30 at a time recommended
>  Paste list of KB IDs into the downloader on the next page
