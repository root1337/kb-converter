Step 5: Upload to KB Website

Upload Instructions

Please Read Carefully

>  Enter the KB to be uploaded in the text box below and open its backend page
>  Navigate to the "Review Options/Comment" section at the bottom of the page
>  Make sure option 3 "In Review" is chosen, enter "HTML" in the box, and click Submit
>  Click "Edit document" button if prompted, then set the editor to HTML mode and clear it
>  Open the converted KB's HTML source code and copy/paste it into the editor
>  Open the KB's attachment folder located below the editor
>  Click "Choose Files", select all of the KB's images, and upload them
>  Click the "Preview" button located right below the editor and select "External View"
>  Make sure KB doesn't look broken and fits Style Guide below
>  Edit HTML manually if something looks wrong
>  Once completed, click Submit and repeat these steps on the next KB

Style Guide

>  Remove any UW header images including the DevHeader if enabled
>  If a title or summary is in the HTML body, move it to the appropriate section outside body
>  Reference the PDF version to make sure the HTML page looks correct

KB Utility Instructions

>  Each utility operates on the KB entered in the text box
>  "KB backend page" opens the admin view backend page of the KB in a web browser
>  "Edit HTML file" opens the HTML file in either Notepad or VS Code
>  "Copy image path" copies the location of the KB's Images folder
    >  This is to make it easier to upload the KB's images to the site
>  Finally, the output folder is where the converted and formatted KBs are stored
